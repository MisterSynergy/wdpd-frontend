About this tool

This tool is developed and maintained by User:MisterSynergy. See also: Wikidata:WikiProject Counter-Vandalism.

If you have suggestions for improvements of this tools, do not hesitate to ask on my user talk page. The source code of this tool is available on Github in two repositories:

  • MisterSynergy/wdpd-frontend
  • MisterSynergy/wdpd-backend

An older version of this tool was available at msbits.toolforge.org/wdcvn/index.php, which itself was an offspring of a PAWS script. This is a newly rewritten frontend for the same content.

Main objectives of this tool, with no particular order, are:

  • Make the patrolling process accessible via statistics. It is extremely helpful to understand the task of patrolling if there is continuous information available about the overall workload and its progress.
  • Provide worklists for user- and item-based patrolling strategies. Since the patrol function effectively is revision-based and thus not ideally suited for Wikidata, other strategies need to be made possible with worklists.
  • General filter options that some other tools also provide.
